# Content Forge ⚒️

**AI‑генератор контента для сайтов**

React SPA, которая генерирует и редактирует контент для сайтов через несколько AI‑провайдеров.  
Фронтенд общается напрямую с API — никакого бэкенда, все данные хранятся в браузере (localStorage).

## Возможности

- **Три провайдера генерации** — OpenAI (GPT‑4o / GPT‑3.5 Turbo), Ollama (локальные LLM), Демо‑режим (без ключей)
- **Генерация контента** — опишите страницу, выберите Tone of Voice, и AI создаст структуру из 3–6 секций
- **Генерация изображений** — DALL‑E 3 (OpenAI) или picsum.photos (демо) для hero, image и gallery секций
- **Редактор секций** — редактируйте текст, меняйте URL изображений, переставляйте секции местами
- **Живое превью** — смотрите, как страница будет выглядеть в финале
- **Экспорт в HTML** — скачайте готовую страницу одним кликом
- **Поиск проектов** — фильтрация по названию на дашборде
- **Настройки** — выбор провайдера, API‑ключ OpenAI, модель, URL и модель Ollama, генерация изображений
- **Статусы проектов** — отслеживание жизненного цикла: черновик → генерация → готов
- **Анимированный фон** — Canvas‑частицы с редкими связями для атмосферы
- **Glass‑дизайн** — единая система стеклянных компонентов с градиентами и анимациями
- **Хранение в браузере** — все проекты и секции сохраняются в localStorage

## Установка и запуск

```bash

npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшна
npm run build

# Предпросмотр собранного проекта
npm run preview
```

## Использование

1. Откройте приложение и перейдите в **Настройки** (⚙️)
2. Выберите провайдер:
   - **OpenAI** — введите API‑ключ (sk-…), выберите модель
   - **Ollama** — укажите URL сервера и модель (должна быть запущена `ollama serve`)
   - **Демо** — ничего не нужно, контент генерируется по шаблонам
3. Включите генерацию изображений (если нужно)
4. Нажмите **+ Новый проект**, укажите название, описание, Tone of Voice и промпт
5. Дождитесь генерации — AI создаст структуру страницы
6. Отредактируйте секции в редакторе: текст, изображения, порядок
7. Посмотрите превью или экспортируйте в HTML

## API

**OpenAI** — требуется API‑ключ с доступом к:
- `gpt-4o` или `gpt-3.5-turbo` для генерации текста
- `dall-e-3` (опционально) для генерации изображений

Получить ключ: [platform.openai.com/api-keys](https://platform.openai.com/api-keys)

**Ollama** — требуется запущенный локальный сервер:
- URL по умолчанию: `http://localhost:11434`
- Любая установленная модель (Llama 3, Mistral, Gemma, Qwen и т.д.)
- Установить: [ollama.com](https://ollama.com)

**Демо** — не требует внешних зависимостей, работает полностью в браузере.
